Ape Lang

Ape is a dynamic, general-purpose programming language.*

* This is a recreational project, so the main goal is having fun, studying language design and exploring various subjects related to it. Please do not use Ape in a production environment unless you know what you are doing.

  • Fast garbage-collected language
  • Dependency-free binary
  • Cross-platform
  • First-class functions

let fibonacci = fn(n) {
  if (n == 0) {
    0
  } else {
    if (n == 1) {
      1
    } else {
      fibonacci(n - 1) + fibonacci(n - 2);
    }
  }
};
